# VisitFloripa — Guia Turístico de Florianópolis

Portal turístico completo da Ilha da Magia: voos em tempo real, praias, hospedagem, gastronomia, entretenimento e roteiros personalizados com IA.

**URL**: https://florianopolisairport.com

## Tecnologias

- Vite + React + TypeScript
- Tailwind CSS + shadcn/ui
- Supabase (banco de dados + Edge Functions)
- Framer Motion
- TanStack React Query

## Desenvolvimento local

```sh
# Clone o repositório
git clone <YOUR_GIT_URL>

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```

O servidor estará disponível em `http://localhost:8080`.

## Deploy

O projeto é publicado via GitHub Actions ou Vercel/Netlify com domínio customizado em `florianopolisairport.com`.
